### Context
* 파이썬 프로그래밍의 가장 기본적인 데이터 관리 단위인 변수(Variable)의 개념과 선언 방식을 학습합니다.
* 데이터를 메모리에 저장하고 호출하는 과정을 이해하여 효율적인 코드 작성의 토대를 마련합니다.

### Core
```python
# 변수 선언 및 데이터 할당
# [변수이름 = 값] 형태
a = 10
b = "Python"

# 변수 출력 (print 함수 활용)
print(a)
print(b)

# 변수 값의 재할당 및 교체
# a = b와 b = a는 서로 다른 의미를 가짐 (오른쪽 값을 왼쪽 변수에 할당)
a = 20
print(a) # 출력: 20

# 동적 타이핑 (Dynamic Typing)
# 변수의 타입을 미리 정하지 않고 할당되는 값에 따라 결정됨
data = 100
data = "텍스트로 변경"
print(data)
```

### Insight
* 파이썬의 변수는 값을 담는 '박스(Box)'보다는 객체(Object)에 붙이는 '이름표(Label)' 또는 '참조(Reference)'에 가깝습니다. 변수가 객체의 메모리 주소를 가리키는 방식이기 때문에, 하나의 데이터를 여러 변수가 동시에 참조할 수도 있습니다.
* 할당 연산자(=)는 '같다'는 의미가 아니라 '대입'을 의미하며, 오른쪽 항의 계산이 끝난 후 왼쪽 변수에 연결됩니다.
* 변수명 작성 시 영문 소문자와 언더바(_)를 사용하는 스네이크 케이스(snake_case)를 권장하며, 숫자로 시작하거나 파이썬 예약어(if, for, print 등)를 변수명으로 사용하는 것은 피해야 합니다.
